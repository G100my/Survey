situation: icon size < 48

提供的 icon size 小於 48x48，

Chrome dev tool 提示 Manifest does not contain a suitable icon.
PNG, SVG or WebP format of at least 144px is required.
the sizes attribute must be set, and the purpose attribute, if set, must include "any" or "maskable".

因為提供的 icon 無法被使用，會將 manifest 判讀為不可安裝，
即使按下 add to Home screen，產出的會是一般書籤，
而不是透過 manifest 產出的 web application。
