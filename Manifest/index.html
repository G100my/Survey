Manifest.json - Test page：

測試範圍：mobile devices

測試項目：

  • display 的 4 種顯示方式
  • icons size 在不同螢幕尺寸設備會由 UA 匹配到什麼尺寸的 icon

使用 Chrome、Firefox等瀏覽器，開啟此 Test page，點擊瀏覽器選單 - add to Home screen。 (有些瀏覽器不必點擊就會跳出 Install prompts)

除了要測試的 member 項目，其他未提及的 manifest member 會以

  • display: standalone
  • name: 完整的測試項目 例：situation: many icons
  • short_name: 測試項目代稱 例：many icons
  • icon: 192、512
的方式呈現。
